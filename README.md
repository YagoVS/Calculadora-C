# Calculadora-C
Este é meu projeto incial na linguagem C, onde desenvolvo uma calculadora bem simples que interage com o usúario e retorna os resultados de suas escolhas.

Projeto simples desenvolvido para praticar e aprimorar conhecimento:

- Variáveis
- Switch case
- Operações matemáticas
- Entrada de dados

## Funionalidade

- Soma
- Subtração
- Divisão
- Multiplicação

## Tecnologias
- Linguagem C
